Pot.Hash.prototype.toJSON

{String} Pot.Hash.prototype.toJSON ()

Pot.js で利用可能。 PotLite.js では利用できません。

Hash を JSON で取得します。

Pot.Hash インスタンスに設定されているすべての要素を、JSON 文字列で取得します。

インスタンスのすべての要素を持つ JSON 文字列が返ります。 

var hash = new Pot.Hash({foo: [1], bar: [2], baz: [3]});
hash.set({a: 4, b: 5, c: [6, 7, '"hoge"']});
var json = hash.toJSON();
Pot.debug(json);
// '{"foo":[1],"bar":[2],"baz":[3],"a"4,"b":5,"c":[6,7,"\"hoge\""]}'
