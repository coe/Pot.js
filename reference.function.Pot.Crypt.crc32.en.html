Pot.Crypt.crc32

{Number} Pot.Crypt.crc32 ({String} string)

Pot.js で利用可能。 PotLite.js では利用できません。

文字列 string の crc32 多項式計算を行います。

引数 string の 32 ビット長の CRC (cyclic redundancy checksum) チェックサムを生成します。

Pot.globalize() が適応されている場合、Pot.crc32() が crc32() で実行できます。

string の crc32 チェックサムの整数値が返ります。 

Pot.debug(Pot.crc32('abc123')); // -821904548

Pot.sprintf() を用いて表記: 

debug( Pot.sprintf('%u',   Pot.crc32('abc123')) ); // '3473062748'
debug( Pot.sprintf('%08x', Pot.crc32('abc123')) ); // 'cf02bb5c'
