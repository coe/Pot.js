Pot.Text.ord

{Number} Pot.Text.ord ({String} s)

Pot.js で利用可能。 PotLite.js では利用できません。

文字列 s の先頭の Unicode 値を返します。

引数 s の、最初の文字の Unicode 値を数値で返します。
String(s).charCodeAt(0) のショートカットとして扱えます。
s が文字列でない場合、Pot.stringify() によって文字列化されます。

Pot.globalize() が適応されている場合、Pot.ord() が ord() で実行できます。

s の最初の文字による Unicode 値が返ります。 

debug(Pot.ord('Hello')); // 72
