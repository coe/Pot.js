Pot.Hash.prototype.clear

{Pot.Hash} Pot.Hash.prototype.clear ()

Pot.js で利用可能。 PotLite.js では利用できません。

Hash の持っているすべての要素をクリアします。

インスタンスが持っているすべての要素を削除し、初期化します。

Pot.Hash インスタンス自身が返ります。 

var hash = new Pot.Hash();
hash.set('foo', {foo: 1});
hash.set('bar', ['bar', 2]);
hash.clear();
Pot.debug(hash.toJSON()); // {}
