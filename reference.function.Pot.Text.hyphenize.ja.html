Pot.Text.hyphenize

{String} Pot.Text.hyphenize ({String} s)

Pot.js で利用可能。 PotLite.js では利用できません。

文字列 s をハイフン記法 (表記) にして返します。

引数 s は、対象の文字列を指定します。
ハイフンで連結した表記にした文字列が返ります。

Pot.globalize() が適応されている場合、Pot.hyphenize() が hyphenize() で実行できます。

s をハイフンで連結した表記にした文字列が返ります。 

var result = Pot.hyphenize('fontSize');
debug(result); // 'font-size'
