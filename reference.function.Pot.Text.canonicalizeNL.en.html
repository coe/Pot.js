Pot.Text.canonicalizeNL

{String} Pot.Text.canonicalizeNL ({String} s)

Pot.js で利用可能。 PotLite.js では利用できません。

改行を \n で統一します。

引数 s の改行コードすべてを \n (LF U+000A) に統一して返します。

Pot.globalize() が適応されている場合、Pot.canonicalizeNL() が canonicalizeNL() で実行できます。

s の改行が \n で統一された文字列が返ります。 

var string = 'foo\r\nbar\rbaz\n';

var result = Pot.canonicalizeNL(string);

debug(result); // 'foo\nbar\nbaz\n'
