Pot.Struct.values

{Array} Pot.Struct.values ({Object} o)

Pot.js で利用可能。 PotLite.js では利用できません。

引数 o の値を配列で取得します。

オブジェクト o の各値を配列で取得します。 Pot.keys() が添字に対して、この関数は値です。

Pot.globalize() が適応されている場合、Pot.values() が values() で実行できます。

o の値のすべてを含んだ配列が返ります。 

var obj = {foo: 1, bar: 2, baz: 3};
Pot.debug(Pot.values(obj)); // [1, 2, 3]

var array = ['foo', 'bar', 'baz'];
Pot.debug(Pot.values(array)); // ['foo', 'bar', 'baz'];

delete array[1];
Pot.debug(Pot.values(array)); // ['foo', 'baz']
