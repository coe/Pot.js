Pot.MimeType.getExtByMimeType

{String} Pot.MimeType.getExtByMimeType ({String} mimeType)

Pot.js で利用可能。 PotLite.js では利用できません。

MIMEType から拡張子を取得します。

'application/javascript' のような MIMEType から、'js' などのファイル拡張子を取得します。
大文字小文字を区別しないで比較し、類似のものがあればそれを返します。
見つからなかった場合は、空文字 ('') を返します。
拡張子は、すべて小文字で返されます。

Pot.globalize() が適応されている場合、Pot.getExtByMimeType() が getExtByMimeType() で実行できます。

拡張子が文字列で返ります。見つからなかった場合は空文字 ('') が返ります。 

Pot.debug( Pot.getExtByMimeType('application/javascript') ); // 'js'
