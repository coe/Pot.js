Pot.Browser.opera

Pot.Browser.opera {Object|Void}

Pot.js と PotLite.js で利用可能。

ブラウザが Opera かどうか。

実行中のブラウザが Opera なら、Pot.Browser.opera がオブジェクトになります。
違う場合、Pot.Browser.opera は undefined になります。
Pot.Browser.opera は version というプロパティを持ちます。
version は文字列で保持されます。
以下の例のように判断し、バージョンを取得できます。 

if (Pot.Browser.opera) {
    debug(Pot.Browser.opera.version);
}

Pot.js は、バージョンの比較に compareVersions() が利用できます。 

if (compareVersions(Pot.Browser.opera.version, '10.5', '<=')) {
    // バージョンが 10.5 以下なら
    // ...
}
