Pot.isBlob

{Boolean} Pot.isBlob ({*} x)

Pot.js と PotLite.js で利用可能。

引数 x が Blob インスタンスかどうかチェックします。

引数 x が Blob インスタンスかどうか調べ、適正なら true を返します。 Blob インスタンスでない場合、false が返ります。

Pot.globalize() が適応されている場合、Pot.isBlob() が isBlob() で実行できます。

引数 x が Blob インスタンスなら true が、そうでない場合 false が返ります。 

var b = Pot.createBlob('hoge');
Pot.debug(Pot.isBlob(blob));   // true
Pot.debug(Pot.isBlob({}));     // false
Pot.debug(Pot.isBlob('hoge')); // false
