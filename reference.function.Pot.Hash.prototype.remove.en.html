Pot.Hash.prototype.remove

{Pot.Hash} Pot.Hash.prototype.remove ({String} key)

Pot.js で利用可能。 PotLite.js では利用できません。

指定のキーに対応する要素を Hash から削除します。

引数 key をキー名として、対応する要素を削除します。

Pot.Hash インスタンス自身が返ります。 

var hash = new Pot.Hash();
hash.set('foo', {bar: 1});
Pot.debug(hash.get('foo')); // {bar: 1}
hash.remove('foo');
Pot.debug(hash.get('foo')); // undefined
