Pot.System.isSafariExtension

Pot.System.isSafariExtension {Boolean|Void}

Pot.js と PotLite.js で利用可能。

Safari Extension で実行されているかどうか。

Safari Extension で実行中なら、Pot.System.isSafariExtension が true になります。
違う場合、Pot.System.isSafariExtension は undefined になります。 

if (Pot.System.isSafariExtension) {
    // Safari Extension で実行中
    // ...
}
