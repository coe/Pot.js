Pot.Text.toKatakanaCase

{String} Pot.Text.toKatakanaCase ({String} text)

Pot.js で利用可能。 PotLite.js では利用できません。

全角ひらがなを全角カタカナに変換します。

引数 text は、変換する文字列を指定します。

Pot.globalize() が適応されている場合、Pot.toKatakanaCase() が toKatakanaCase() で実行できます。

全角ひらがなを全角カタカナに変換した文字列が返ります。 

debug(Pot.toKatakanaCase('ぼぽう゛ぁあぃいぅうぇえぉお'));
// 'ボポヴァアィイゥウェエォオ'
