Pot.ArrayBufferoid.prototype.toUint8ClampedArray

{Uint8ClampedArray|Array} Pot.ArrayBufferoid.prototype.toUint8ClampedArray ()

Pot.js で利用可能。 PotLite.js では利用できません。

Uint8ClampedArray に変換します。

Pot.ArrayBufferoid 自身が持っている要素と共に Uint8ClampedArray に変換した新しいオブジェクトを返します。

変換された Uint8ClampedArray または Array が返ります。 

var bufferoid = new Pot.ArrayBufferoid([1, 2, 3]);
var uint8clampedArray = bufferoid.toUint8ClampedArray();
Pot.debug(uint8clampedArray); // [1, 2, 3]
