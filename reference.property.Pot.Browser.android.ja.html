Pot.Browser.android

Pot.Browser.android {Object|Void}

Pot.js と PotLite.js で利用可能。

実行環境が Android かどうか。

実行中の環境が Android なら、Pot.Browser.android がオブジェクトになります。
違う場合、Pot.Browser.android は undefined になります。
Pot.Browser.android は version というプロパティを持ちます。
version は文字列で保持されます。
以下の例のように判断し、バージョンを取得できます。 

if (Pot.Browser.android) {
    debug(Pot.Browser.android.version);
}

Pot.js は、バージョンの比較に compareVersions() が利用できます。 

if (compareVersions(Pot.Browser.android.version, '4.0', '>=')) {
    // バージョンが 4.0 以上なら
    // ...
}
