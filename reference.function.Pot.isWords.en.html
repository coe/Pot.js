Pot.isWords

{Boolean} Pot.isWords ({String} token)

Pot.js と PotLite.js で利用可能。

引数 token が words かどうか調べます。

引数 token をトークンまたは単一の文字列とみなし、それが words であれば true を返します。
words は、添字や変数名などで使用できる範囲の Unicode 値として扱われます。

Pot.globalize() が適応されている場合、Pot.isWords() が isWords() で実行できます。

words であれば true が返ります。違う場合は false が返ります。 

debug(isWords(' '));     // false
debug(isWords('abc'));   // true
debug(isWords('ほげ'));  // true
debug(isWords('\r\n'));  // false
debug(isWords(' \n'));   // false
debug(isWords(' abc'));  // false
debug(isWords('abc '));  // false
debug(isWords('_'));     // true
debug(isWords(false));   // false
debug(isWords(true));    // false
debug(isWords(void 0));  // false
debug(isWords({}));      // false
debug(isWords(['ABC'])); // false
debug(isWords('$hoge')); // true
debug(isWords('$_'));    // true
