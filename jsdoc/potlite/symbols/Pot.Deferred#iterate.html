Pot.Deferred#iterate(callback, (context))
Iterate an iterable object that is previous chain result. Iteration will use the results of the previous chain.
Parameters:
{Function} callback
An iterable function. function(value, key, object) this == `context`. Throw Pot.StopIteration if you want to stop the loop.
{Object} (context)
Optionally, context object. (i.e. this)
Returns:
{Deferred} Return the Deferred.
