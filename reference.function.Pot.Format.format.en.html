Pot.Format.format

{String} Pot.Format.format ({String} tpl [, {*} arg1 [, {*} arg2 [, ...]]])

Pot.js で利用可能。 PotLite.js では利用できません。

それぞれの引数をフォーマットした文字列を返します。

引数 tpl をテンプレートとし、それぞれの引数をフォーマットして返します。
#1 は tpl の次の引数, #2 はその次の引数, ... #10 は ... という変換がなされます。

Pot.globalize() が適応されている場合、Pot.format() が format() で実行できます。

フォーマットされた文字列が返ります。 

var result = Pot.format('#1 + #2 + #3', 10, 20, 30);
debug(result); // '10 + 20 + 30'
var result = Pot.format('J#1v#1#2 ECMA#2', 'a', 'Script');
debug(result); // 'JavaScript ECMAScript'
