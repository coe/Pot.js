Pot.ArrayBufferoid.prototype.size

{Number} Pot.ArrayBufferoid.prototype.size ()

Pot.js で利用可能。 PotLite.js では利用できません。

インスタンスの要素数 (length) を取得します。

対象の Pot.ArrayBufferoid インスタンスの要素数 (length) を取得します。
Pot.ArrayBufferoid は、length の代わりに size() を使用します。

インスタンスの要素数 (length) が返ります。 

var bufferoid = new Pot.ArrayBufferoid([1, 2, 3]);
Pot.debug(bufferoid.size()); // 3
