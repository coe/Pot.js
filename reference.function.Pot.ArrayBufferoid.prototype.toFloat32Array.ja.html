Pot.ArrayBufferoid.prototype.toFloat32Array

{Float32Array|Array} Pot.ArrayBufferoid.prototype.toFloat32Array ()

Pot.js で利用可能。 PotLite.js では利用できません。

Float32Array に変換します。

Pot.ArrayBufferoid 自身が持っている要素と共に Float32Array に変換した新しいオブジェクトを返します。

変換された Float32Array または Array が返ります。 

var bufferoid = new Pot.ArrayBufferoid([1, 2, 3]);
var float32array = bufferoid.toFloat32Array();
Pot.debug(float32array); // [1, 2, 3]
