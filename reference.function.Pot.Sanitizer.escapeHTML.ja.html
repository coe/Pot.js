Pot.Sanitizer.escapeHTML

{String} Pot.Sanitizer.escapeHTML ({String} text)

Pot.js で利用可能。 PotLite.js では利用できません。

引数 text を HTML 文字列としてエスケープします。

引数 text を HTML 文字列とし、エスケープします。

  • & は &amp; になります。
  • < は &lt; になります。
  • > は &gt; になります。
  • " は &quot; になります。
  • ' は &#039; になります。

この関数は、既にエスケープされている文字列を 2 重でエスケープしません。

Pot.globalize() が適応されている場合、Pot.escapeHTML() が escapeHTML() で実行できます。

エスケープされた文字列が返ります。 

var string = '(>_<)/"< Hello World!';
var result = Pot.escapeHTML(string);
Pot.debug(result);
// '(&gt;_&lt;)/&quot;&lt; Hello World!'
