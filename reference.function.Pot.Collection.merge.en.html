Pot.Collection.merge

{Array|*} Pot.Collection.merge ({Array|*} args1 [, {Array|*} args2 [, ...]])

Pot.js で利用可能。 PotLite.js では利用できません。

与えられた配列を結合して返します。

引数で与えられた配列をすべて結合し、新しい配列として返します。
与えられた引数自体は変化しません。

Pot.globalize() が適応されている場合、Pot.merge() が merge() で実行できます。

結合された新しい配列が返ります。 

var array1 = [1, 2, 3];
var array2 = [4, 5, 6];
var result = Pot.merge(array1, array2);
Pot.debug(result); // [1, 2, 3, 4, 5, 6]

result = Pot.merge([], 1, 2, 'foo', {bar: 3});
Pot.debug(result); // [1, 2, 'foo', {bar: 3}]
