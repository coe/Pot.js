Pot.Text.stripTags

{String} Pot.Text.stripTags ({String} text)

Pot.js で利用可能。 PotLite.js では利用できません。

文字列 text の HTML タグをすべて除去して返します。

引数 text は、対象の HTML 文字列を渡します。
text 内の HTML タグがすべて除去された文字列が返ります。
コメント (<!-- ... -->) や <? ... ?>, <% ... %>, <![CDATA[ ... ]]> などの要素も削除されます。

Pot.globalize() が適応されている場合、Pot.stripTags() が stripTags() で実行できます。

text の HTML タグがすべて除去された文字列が返ります。 

html = '<div>Hoge</div>foo<br>bar<i>baz</i>';
var result = Pot.stripTags(html);
debug(result); // 'Hoge foo bar baz'
