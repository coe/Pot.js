Pot.isTypedArray

{Boolean} Pot.isTypedArray ({*} x)

Pot.js と PotLite.js で利用可能。

引数 x が TypedArray かどうかチェックします。

引数 x が TypedArray かどうか調べ、適正なら true を返します。 TypedArray でない場合、false が返ります。

Pot.globalize() が適応されている場合、Pot.isTypedArray() が isTypedArray() で実行できます。

引数 x が TypedArray なら true が返り、そうでない場合 false が返ります。 

var obj = {foo : 1};
var arr = [1, 2, 3];
var buf = new ArrayBuffer(10);
var uar = new Uint8Array(10);
Pot.debug(Pot.isTypedArray(obj)); // false
Pot.debug(Pot.isTypedArray(arr)); // false
Pot.debug(Pot.isTypedArray(buf)); // true
Pot.debug(Pot.isTypedArray(uar)); // true
