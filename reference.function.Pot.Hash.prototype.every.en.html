Pot.Hash.prototype.every

{Boolean} Pot.Hash.prototype.every ({Function} callback [, {*} context])

Pot.js で利用可能。 PotLite.js では利用できません。

Hash に対して every イテレートを実行します。

Hash インスタンスの要素に対して every イテレートを実行します。
詳細は、every を参照ください。

結果の真偽値が返ります。 

 var hash = new Pot.Hash();
hash.set({A: 12, B: 5, C: 8, D: 130, E: 44});
var result = hash.every(function(value, key, object) {
    return (value >= 10);
});
Pot.debug(result); // false

hash.clear();
hash.set({A: 12, B: 54, C: 18, D: 130, E: 44});
result = hash.every(function(value, key, object) {
    return (value >= 10);
});
Pot.debug(result); // true
