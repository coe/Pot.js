Pot.Text.strip

{String} Pot.Text.strip ({String} s [, {String} chars [, {Boolean} ignoreCase = false]])

Pot.js で利用可能。 PotLite.js では利用できません。

文字列 s のすべてのホワイトスペースを除去した文字列を返します。

引数 s のすべてのホワイトスペースを除去して返します。
ホワイトスペースは、全角スペース U+3000 を含む Unicode Whitespace キャラクタです。
文字列 chars が指定されると、ホワイトスペースの代わりに chars を除去して返します。
その際に、引数 ignoreCase が 真 (true) として渡されると、大文字小文字を無視して除去します。
chars は、1 文字単位で扱われます。
Pot.trim(), Pot.ltrim(), Pot.rtrim() も参照ください。

Pot.globalize() が適応されている場合、Pot.strip() が strip() で実行できます。

s のすべてのホワイトスペースまたは chars を除去した文字列が返ります。 

var s = '   Hello  World ! \r\n.';

debug(Pot.strip(s)); // 'HelloWorld!.'
var s = 'foo looooooooool';

debug(Pot.strip(s, 'o')); // 'f ll'
