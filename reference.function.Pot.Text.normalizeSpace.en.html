Pot.Text.normalizeSpace

{String} Pot.Text.normalizeSpace ({String} s [, {String} spacer = ' '])

Pot.js で利用可能。 PotLite.js では利用できません。

文字列 s のすべてのホワイトスペースを 1 つのスペースに変換して返します。

引数 s の すべてのホワイトスペースを 1 つのスペース ' ' (U+0020) に変換して返します。
引数 spacer が渡されると、スペースの代わりに spacer に変換します。

Pot.globalize() が適応されている場合、Pot.normalizeSpace() が normalizeSpace() で実行できます。

s のホワイトスペースが spacer に変換された文字列が返ります。 

var s = 'Hello  \r\n  World  \r\n  \t !';

debug(Pot.normalizeSpace(s)); // 'Hello World !'
var s = 'foo       bar     baz';

debug(Pot.normalizeSpace(s, '+')); // 'foo+bar+baz'
