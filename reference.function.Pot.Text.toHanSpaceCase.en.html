Pot.Text.toHanSpaceCase

{String} Pot.Text.toHanSpaceCase ({String} text)

Pot.js で利用可能。 PotLite.js では利用できません。

全角スペース (U+3000) を半角スペース (U+0020) に変換します。

引数 text は、変換する文字列を指定します。

Pot.globalize() が適応されている場合、Pot.toHanSpaceCase() が toHanSpaceCase() で実行できます。

全角スペースを半角スペースに変換した文字列が返ります。 

debug(Pot.toHanSpaceCase('   hoge   ')); // '   hoge   '
