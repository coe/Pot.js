Pot.System.isBuiltinArrayIndexOf

Pot.System.isBuiltinArrayIndexOf {Boolean|Void}

Pot.js と PotLite.js で利用可能。

Array.prototype.indexOf がビルトインでサポートされているかどうか。

Array.prototype.indexOf が、
ビルトインでサポートされている場合 Pot.System.isBuiltinArrayIndexOf が true になります。
サポートされていない場合、false になります。
実行後に Array.prototype.indexOf = function() {...} と定義しても false になります。
Array.prototype.indexOf は、JavaScript 1.6 以降でサポートされます。 

if (Pot.System.isBuiltinArrayIndexOf) {

    // Array.prototype.indexOf がサポート済み
    debug( [1, 2, 3].indexOf(2) ); // 1

}
