Pot.Text.splitBySpace

{Array} Pot.Text.splitBySpace ({String} s)

Pot.js で利用可能。 PotLite.js では利用できません。

文字列 s をホワイトスペースで分割した配列を返します。

引数 s を、ホワイトスペースで分割し 得られた配列を返します。
分割した際に、長さが 0 の要素は除外されます。

Pot.globalize() が適応されている場合、Pot.splitBySpace() が splitBySpace() で実行できます。

分割された配列が返ります。 

var s = '  Hello  \r\n  World  \r\n  \t !  ';

debug(Pot.splitBySpace(s)); // ['Hello', 'World', '!']
