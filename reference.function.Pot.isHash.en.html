Pot.isHash

{Boolean} Pot.isHash ({*} x)

Pot.js で利用可能。 PotLite.js では利用できません。

引数 x が Pot.Hash インスタンスかどうかチェックします。

引数 x が Pot.Hash インスタンスかどうか調べ、適正なら true を返します。 そうでない場合、false が返ります。

Pot.globalize() が適応されている場合、Pot.isHash() が isHash() で実行できます。

引数 x が Pot.Hash インスタンスなら true が返り、そうでない場合 false が返ります。 

var obj = {hoge: 1};
var hash = new Pot.Hash();
Pot.debug(Pot.isHash(obj));  // false
Pot.debug(Pot.isHash(hash)); // true
