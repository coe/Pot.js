Pot.Sanitizer.unescapeSequence

{String} Pot.Sanitizer.unescapeSequence ({String} text)

Pot.js で利用可能。 PotLite.js では利用できません。

JavaScript エスケープシーケンスをアンエスケープします。

引数 text に含まれる JavaScript エスケープシーケンスをアンエスケープして返します。
JSON 文字列のように変換されます。

Pot.globalize() が適応されている場合、Pot.unescapeSequence() が unescapeSequence() で実行できます。

アンエスケープされた文字列が返ります。 

var string = '\\u307b\\u3052abc\\u0020(\\"\\uff57\\")';
var result = Pot.unescapeSequence(string);
Pot.debug(result); // 'ほげabc ("ｗ")'
