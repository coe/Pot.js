Pot.isFileReader

{Boolean} Pot.isFileReader ({*} x)

Pot.js と PotLite.js で利用可能。

引数 x が FileReader インスタンスかどうかチェックします。

引数 x が FileReader インスタンスかどうか調べ、適正なら true を返します。 FileReader インスタンスでない場合、false が返ります。

Pot.globalize() が適応されている場合、Pot.isFileReader() が isFileReader() で実行できます。

引数 x が FileReader インスタンスなら true が返り、そうでない場合 false が返ります。 

var object = {hoge : 1};
var reader = new FileReader();
Pot.debug(Pot.isFileReader(object)); // false
Pot.debug(Pot.isFileReader(reader)); // true
