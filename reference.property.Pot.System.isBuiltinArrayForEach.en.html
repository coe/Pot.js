Pot.System.isBuiltinArrayForEach

Pot.System.isBuiltinArrayForEach {Boolean|Void}

Pot.js と PotLite.js で利用可能。

Array.prototype.forEach がビルトインでサポートされているかどうか。

Array.prototype.forEach が、
ビルトインでサポートされている場合 Pot.System.isBuiltinArrayForEach が true になります。
サポートされていない場合、false になります。
実行後に Array.prototype.forEach = function() {...} と定義しても false になります。
Array.prototype.forEach は JavaScript 1.6 以降 でサポートされます。 

if (Pot.System.isBuiltinArrayForEach) {

    // Array.prototype.forEach がサポート済み
    [1, 2, 3].forEach(function(val) {
        // ...
    });

}
