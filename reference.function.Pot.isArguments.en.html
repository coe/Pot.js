Pot.isArguments

{Boolean} Pot.isArguments ({*} x)

Pot.js と PotLite.js で利用可能。

引数 x が arguments かどうかチェックします。

引数 x が arguments かどうか調べ、適正なら true を返します。 arguments でない場合、false が返ります。

Pot.globalize() が適応されている場合、Pot.isArguments() が isArguments() で実行できます。

引数 x が arguments なら true が返り、そうでない場合 false が返ります。 

(function(a, b, c) {
    var obj = {foo : 1};
    var arr = [1, 2, 3];
    Pot.debug(Pot.isArguments(obj));       // false
    Pot.debug(Pot.isArguments(arr));       // false
    Pot.debug(Pot.isArguments(arguments)); // true
}(1, 2, 3));
