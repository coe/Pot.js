Pot.Struct.flip

{Object|*} Pot.Struct.flip ({Object|Array|*} o)

Pot.js で利用可能。 PotLite.js では利用できません。

o の値を反転させて返します。

引数 o の値 (キー) を反転させたコピーを取得します。

Pot.globalize() が適応されている場合、Pot.flip() が flip() で実行できます。

値を反転させた o のコピーが返ります。 

Pot.debug(Pot.flip({foo: 'A', bar: 'B', baz: 'C'}));
// {A: 'foo', B: 'bar', C: 'baz'}
