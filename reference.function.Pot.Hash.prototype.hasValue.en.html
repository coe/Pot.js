Pot.Hash.prototype.hasValue

{Boolean} Pot.Hash.prototype.hasValue ({*} value)

Pot.js で利用可能。 PotLite.js では利用できません。

Hash に指定の値が存在するか調べます。

引数 value が存在するかどうかを真偽値で取得します。
比較は === によって行われます。
1 つでも value と同じ値があった場合、true が返ります。

値が存在する場合は true が返り、存在しなかった場合は false が返ります。 

var hash = new Pot.Hash();
hash.set('foo', 1);
hash.set('bar', 'abc');
hash.set('baz', [1, 2, 3]);
Pot.debug(hash.hasValue(1));     // true
Pot.debug(hash.hasValue('1'));   // false
Pot.debug(hash.hasValue('abc')); // true
