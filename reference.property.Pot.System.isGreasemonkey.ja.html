Pot.System.isGreasemonkey

Pot.System.isGreasemonkey {Boolean|Void}

Pot.js と PotLite.js で利用可能。

Greasemonkey で実行されているかどうか。

Greasemoneky (userscript) で実行中なら、Pot.System.isGreasemonkey が true になります。
違う場合、Pot.System.isGreasemonkey は undefined になります。 

if (Pot.System.isGreasemonkey) {
    // Greasemonkey で実行中
    // ...
}
