Pot.ArrayBufferoid.prototype.seek

{Number} Pot.ArrayBufferoid.prototype.seek ({Number} offset)

Pot.js で利用可能。 PotLite.js では利用できません。

ストリーム中の位置を offset に移動します。

Pot.ArrayBufferoid をストリームとして利用する際の、位置を設定します。
引数 offset は、移動する位置を整数で指定します。

移動後の位置 (オフセット) が返ります。 

var buffer = new Pot.ArrayBufferoid([1, 2, 3, 4, 5]);
Pot.debug(buffer.tell());  // 0
buffer.seek(1);
Pot.debug(buffer.tell());  // 1
Pot.debug(buffer.read(1)); // [2]
buffer.seek(3);
Pot.debug(buffer.tell());  // 3
Pot.debug(buffer.read(2)); // [4, 5]
