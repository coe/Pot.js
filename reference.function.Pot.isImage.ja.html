Pot.isImage

{Boolean} Pot.isImage ({*} x)

Pot.js と PotLite.js で利用可能。

引数 x が Image インスタンスかどうかチェックします。

引数 x が Image インスタンスかどうか調べ、適正なら true を返します。 Image インスタンスでない場合、false が返ります。

Pot.globalize() が適応されている場合、Pot.isImage() が isImage() で実行できます。

引数 x が Image インスタンスなら true が返り、そうでない場合 false が返ります。 

var object = {hoge : 1};
var image = new Image();
Pot.debug(Pot.isImage(object)); // false
Pot.debug(Pot.isImage(image));  // true
