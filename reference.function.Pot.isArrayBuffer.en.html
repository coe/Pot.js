Pot.isArrayBuffer

{Boolean} Pot.isArrayBuffer ({*} x)

Pot.js と PotLite.js で利用可能。

引数 x が ArrayBuffer かどうかチェックします。

引数 x が ArrayBuffer インスタンスかどうか調べ、適正なら true を返します。 そうでない場合、false が返ります。

Pot.globalize() が適応されている場合、Pot.isArrayBuffer() が isArrayBuffer() で実行できます。

引数 x が ArrayBuffer インスタンスなら true が返り、そうでない場合 false が返ります。 

var obj = {foo : 1};
var arr = [1, 2, 3];
var buf = new ArrayBuffer(10);
var uar = new Uint8Array(10);
Pot.debug(Pot.isArrayBuffer(obj)); // false
Pot.debug(Pot.isArrayBuffer(arr)); // false
Pot.debug(Pot.isArrayBuffer(buf)); // true
Pot.debug(Pot.isArrayBuffer(uar)); // false
