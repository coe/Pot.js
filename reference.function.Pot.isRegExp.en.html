Pot.isRegExp

{Boolean} Pot.isRegExp ({*} x)

Pot.js と PotLite.js で利用可能。

引数 x が正規表現オブジェクトかどうかチェックします。

引数 x が正規表現オブジェクトかどうか調べ、適正なら true を返します。
正規表現オブジェクトでない場合、false が返ります。

x が正規表現オブジェクトなら true を返します。 

debug(isRegExp(/fo+/));              // true
debug(isRegExp(new RegExp('fo+$'))); // true
debug(isRegExp('hoge'));             // false
