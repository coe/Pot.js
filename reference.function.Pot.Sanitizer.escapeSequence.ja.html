Pot.Sanitizer.escapeSequence

{String} Pot.Sanitizer.escapeSequence ({String} text)

Pot.js で利用可能。 PotLite.js では利用できません。

JavaScript エスケープシーケンスとしてエスケープします。

エスケープは JSON 文字列のように変換されます。

Pot.globalize() が適応されている場合、Pot.escapeSequence() が escapeSequence() で実行できます。

エスケープされた文字列が返ります。 

var string = 'ほげabc ("ｗ")';
var result = Pot.escapeSequence(string);
Pot.debug(result);
// '\u307b\u3052abc\u0020(\"\uff57\")'
