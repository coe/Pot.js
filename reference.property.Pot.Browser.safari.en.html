Pot.Browser.safari

Pot.Browser.safari {Object|Void}

Pot.js と PotLite.js で利用可能。

ブラウザが Safari かどうか。

実行中のブラウザが Safari なら、Pot.Browser.safari がオブジェクトになります。
違う場合、Pot.Browser.safari は undefined になります。
Pot.Browser.safari は version というプロパティを持ちます。
version は文字列で保持されます。
以下の例のように判断し、バージョンを取得できます。 

if (Pot.Browser.safari) {
    debug(Pot.Browser.safari.version);
}

Pot.js は、バージョンの比較に compareVersions() が利用できます。 

if (compareVersions(Pot.Browser.safari.version, '5', '<')) {
    // バージョン 5 以前なら
    // ...
}
