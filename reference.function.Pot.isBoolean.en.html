Pot.isBoolean

{Boolean} Pot.isBoolean ({*} x)

Pot.js と PotLite.js で利用可能。

引数 x が真偽値かどうかチェックします。

引数 x が真偽値かどうか調べ、適正なら true を返します。
真偽値でない場合、false が返ります。

x が真偽値であれば true を返します。 

debug(isBoolean(true));               // true
debug(isBoolean(new Boolean(false))); // true
debug(isBoolean('hoge'));             // false
