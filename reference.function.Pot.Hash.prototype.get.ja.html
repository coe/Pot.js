Pot.Hash.prototype.get

{*} Pot.Hash.prototype.get ({String} key)

Pot.js で利用可能。 PotLite.js では利用できません。

キー名を指定して Hash から値を取得します。

引数 key は、取得するキー名を指定します。
キーが存在しなかった場合は undefined が返ります。

取得した値が返ります。 

var hash = new Pot.Hash();
hash.set('foo', 1);
Pot.debug(hash.get('foo'));  // 1
Pot.debug(hash.get('hoge')); // undefined
