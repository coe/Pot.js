Pot.Text.chr

{String} Pot.Text.chr ({Number|Array} ...args [, {Number} ...args1 [, ...]])

Pot.js で利用可能。 PotLite.js では利用できません。

Unicode の値の指定のシーケンスを使用することによって生成された文字列を返します。

Pot.chr() は、String.fromCharCode とほとんど同じです。
違う点は、String.fromCharCode.apply(null, new Array(100000000)); のような
巨大な引数を渡した時に、環境により発生するスタックオーバーフローを回避します。
また、配列が渡された場合は apply のように実行します。
それ以外は同じです。String.fromCharCode と同じように使用できます。

Pot.globalize() が適応されている場合、Pot.chr() が chr() で実行できます。

Unicode の値の指定のシーケンスによって生成された文字列を返します。 

debug(Pot.chr(97)); // 'a'
debug(Pot.chr(97, 98, 99)); // 'abc'
debug(
    Pot.chr(
        [72, 101, 108, 108, 111, 32, 87, 111, 114, 108, 100, 33]
    )
); // 'Hello World!'
