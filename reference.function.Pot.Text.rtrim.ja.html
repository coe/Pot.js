Pot.Text.rtrim

{String} Pot.Text.rtrim ({String} s [, {String} chars [, {Boolean} ignoreCase = false]])

Pot.js で利用可能。 PotLite.js では利用できません。

文字列 s の右側のホワイトスペースを除去した文字列を返します。

引数 s の右側のホワイトスペースを除去して返します。
ホワイトスペースは、全角スペース U+3000 を含む Unicode Whitespace キャラクタです。
文字列 chars が指定されると、ホワイトスペースの代わりに chars を除去して返します。
その際に、引数 ignoreCase が 真 (true) として渡されると、大文字小文字を無視して除去します。
chars は、1 文字単位で扱われます。
Pot.trim(), Pot.ltrim() も参照ください。

Pot.globalize() が適応されている場合、Pot.rtrim() が rtrim() で実行できます。

s の右側のホワイトスペースまたは chars を除去した文字列が返ります。 

debug( Pot.rtrim(' hoge  ') );     // ' hoge'

debug( Pot.rtrim('abc a', 'ac') ); // 'abc '
