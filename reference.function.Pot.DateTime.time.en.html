Pot.DateTime.time

{Number} Pot.DateTime.time ()

Pot.js で利用可能。 PotLite.js では利用できません。

現在の UNIX 時間を取得します。

UNIX 時間 (UTC での 1970 年 1 月 1 日 0 時 0 分 0 秒 からの経過秒数) を取得します。
返る数値はミリ秒ではなく 秒数になります。

Pot.globalize() が適応されている場合、Pot.time() が time() で実行できます。

現在の UNIX 時間が 数値で返ります。 

debug( time() ); // e.g. 1331086567
