Pot.ArrayBufferoid.prototype.toArrayBuffer

{ArrayBuffer|Array} Pot.ArrayBufferoid.prototype.toArrayBuffer ()

Pot.js で利用可能。 PotLite.js では利用できません。

ArrayBuffer に変換します。

Pot.ArrayBufferoid 自身が持っている要素と共に ArrayBuffer に変換した新しいオブジェクトを返します。

変換された ArrayBuffer または Array が返ります。 

var bufferoid = new Pot.ArrayBufferoid([1, 2, 3]);
var buffer = bufferoid.toArrayBuffer();
Pot.debug(new Uint8Array(buffer)); // [1, 2, 3]
