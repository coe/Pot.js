Pot.Hash.prototype.map

{Pot.Hash} Pot.Hash.prototype.map ({Function} callback [, {*} context])

Pot.js で利用可能。 PotLite.js では利用できません。

Hash に対して map イテレートを実行します。

Hash インスタンスの要素に対して map イテレートを実行します。
詳細は、map を参照ください。

新しい Pot.Hash インスタンスが返ります。 

var hash = new Pot.Hash();
hash.set({a: 1, b: 2, c: 3, d: 4, e: 5});
var result = hash.map(function(value, key) {
    return value * 100;
});
Pot.debug(result.toObject());
// {a: 100, b: 200, c: 300, d: 400, e: 500}
