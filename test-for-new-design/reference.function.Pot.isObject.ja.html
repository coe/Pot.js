Pot.isObject

{Boolean} Pot.isObject ({*} x)

Pot.js と PotLite.js で利用可能。

引数 x がオブジェクトかどうかチェックします。

引数 x がオブジェクトかどうか調べ、適正なら true を返します。
オブジェクトでない場合、false が返ります。

x がオブジェクトなら true を返します。 

debug(isObject({}));           // true
debug(isObject({a: 1, b: 2})); // true
debug(isObject(new Object())); // true
debug(isObject([100]));        // false
debug(isObject('hoge'));       // false
