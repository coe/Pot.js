{Boolean} Pot.isNumber ({*} x)

Pot.js と PotLite.js で利用可能。

引数 x が数値かどうかチェックします。

引数 x が数値かどうか調べ、適正なら true を返します。
数値でない場合、false が返ります。

x が数値であれば true を返します。 

debug(isNumber(0));               // true
debug(isNumber(new Number(123))); // true
debug(isNumber(NaN));             // true
debug(isNumber(Infinity));        // true
debug(isNumber('hoge'));          // false
