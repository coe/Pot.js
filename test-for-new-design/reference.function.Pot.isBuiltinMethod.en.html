{Boolean} Pot.isBuiltinMethod ({Function|*} method)

Pot.js と PotLite.js で利用可能。

引数 method がビルトイン関数かどうかチェックします。

引数 method がビルトイン関数かどうか調べます。
例えば encodeURI を渡すと true を返します。

method がビルトイン関数なら true を返し、それ以外は false を返します。 

debug(isBuiltinMethod((function() {})));          // false
debug(isBuiltinMethod(Array.prototype.slice));    // true
debug(isBuiltinMethod(encodeURIComponent));       // true
debug(isBuiltinMethod(document.getElementById));  // true (on Web Browser)
