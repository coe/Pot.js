{Boolean} Pot.isFunction ({*} x)

Pot.js と PotLite.js で利用可能。

引数 x が関数オブジェクトかどうかチェックします。

引数 x が関数オブジェクトかどうか調べ、適正なら true を返します。
関数でない場合、false が返ります。

x が関数なら true を返します。 

debug(isFunction((function(){})));           // true
debug(isFunction(new Function('return 1'))); // true
debug(isFunction('hoge'));                   // false
