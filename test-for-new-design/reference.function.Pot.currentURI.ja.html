Pot.currentURI

{String} Pot.currentURI ()

Pot.js と PotLite.js で利用可能。

現在の URI もしくはファイル名を取得します。

Pot.js もしくは PotLite.js 読み込み時の URI、または実行中のファイル名を返します。
基本的に location.href の値と同じです。
Node.js などの非ブラウザ環境では、実行中のファイル名が返ります。
canonical などの影響により URI が取得できなかった場合は、空文字 '' を返します。
複数の window が存在する環境 (XUL など) で効果が現れます。
返す値は、読み込み時以降 違う URI になったりせず、変化することはありません。

現在の URI またはファイル名、もしくは空文字 '' が返ります。 

// 例えば http://www.example.com/hoge.html 上で実行中の場合

var uri = currentURI();
debug(uri); // 'http://www.example.com/hoge.html'
