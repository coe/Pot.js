Pot.Deferred.prototype.begin

{Pot.Deferred} Pot.Deferred.prototype.begin ([{*} ...args])

Pot.js と PotLite.js で利用可能。

Deferred チェインを開始します。

コールバックチェインを開始します。
任意に、引数に値を持たせ 次のコールバック関数の引数に渡すことができます。

実行時の Pot.Deferred インスタンスが返ります。 

var d = new Pot.Deferred();

d.then(function() {
    debug(1);
    return 1;
});
// この時点ではチェインは実行されていない

// なんらかの条件が通ればコールバック関数を追加
if (check()) {
    d.then(function(res) {
        return someFunc(res);
    });
}
// チェインを実行
d.begin();

// その後は追加した時に実行される
d.then(function(res) {
    return 'hoge';
});
d.then(function(res) {
    debug(res); // 'hoge'
});
