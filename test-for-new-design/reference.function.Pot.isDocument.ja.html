{Boolean} Pot.isDocument ({Document|Element|Node|*} x)

Pot.js と PotLite.js で利用可能。

引数 x が DOM Document オブジェクトかどうかチェックします。

引数 x が DOM Document オブジェクトかどうか調べます。

x が document オブジェクトなら true を返し、それ以外は false を返します。 

// For Web Browser.
debug(isDocument(window));        // false
debug(isDocument(document));      // true
debug(isDocument(document.body)); // false
