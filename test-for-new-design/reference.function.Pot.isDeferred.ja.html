{Boolean} Pot.isDeferred ({*} x)

Pot.js と PotLite.js で利用可能。

引数 x が Pot.Deferred のインスタンスかどうかチェックします。

引数 x が Pot.Deferred のインスタンスかどうか調べます。

x が Pot.Deferred のインスタンスなら true を返し、それ以外は false を返します。 

var o = {hoge: 1};
var d = new Pot.Deferred();

debug( isDeferred(o) ); // false
debug( isDeferred(d) ); // true
