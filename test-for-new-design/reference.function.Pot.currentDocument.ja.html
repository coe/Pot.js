{Object} Pot.currentDocument ()

Pot.js と PotLite.js で利用可能。

現在の document オブジェクトを取得します。

Pot.js もしくは PotLite.js 読み込み時の document オブジェクトを返します。
document が存在しない場合は 空のオブジェクト {} を返します。
複数の window や document が存在する環境 (XUL など) で効果が現れます。
返す値は、読み込み時以降 違う document オブジェクトになったりせず、変化することはありません。

現在の document オブジェクトもしくは空のオブジェクト {} が返ります。 

var doc = currentDocument();
debug( doc.getElementById('hoge').tagName ); // e.g., 'DIV'
