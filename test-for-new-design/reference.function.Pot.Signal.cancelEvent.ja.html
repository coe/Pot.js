Pot.Signal.cancelEvent

{Boolean} Pot.Signal.cancelEvent ({Event|Object|*} ev)

Pot.js と PotLite.js で利用可能。

引数で与えられたイベントをキャンセルします。

引数で与えられた Event オブジェクトを中止します。
この関数は、 

ev.preventDefault();
ev.stopPropagation();

とするのと同じ働きをします。
また、常に false を返すので
jQuery イベント内などで return Pot.cancelEvent(ev); と返すことで false 扱いもできます。
コードを短くする用途などに活用できます。

Pot.globalize() が適応されている場合、Pot.cancelEvent() が cancelEvent() で実行できます。

常に false を返します。 

Pot.attach('#foo', 'click', function(ev) {
    myProcess();
    // イベントが伝達しないようキャンセル
    return cancelEvent(ev);
});
