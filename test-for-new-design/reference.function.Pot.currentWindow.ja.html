{Object} Pot.currentWindow ()

Pot.js と PotLite.js で利用可能。

現在の window オブジェクトを取得します。

Pot.js もしくは PotLite.js 読み込み時の window オブジェクトを返します。
window が存在しない場合は 空のオブジェクト {} を返します。
複数の window が存在する環境 (XUL など) で効果が現れます。
返す値は、読み込み時以降 違う window オブジェクトになったりせず、変化することはありません。

現在の window オブジェクトもしくは空のオブジェクト {} が返ります。 

var win = currentWindow();
win.alert('hoge'); // 'hoge'
