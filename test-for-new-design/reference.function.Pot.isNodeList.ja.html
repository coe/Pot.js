{Boolean} Pot.isNodeList ({*} x)

Pot.js と PotLite.js で利用可能。

引数 x が DOM NodeList かどうかチェックします。

引数 x が DOM NodeList かどうか調べます。

x が NodeList なら true を返し、それ以外は false を返します。 

// For Web Browser.
var array = [{foo: 1}, {bar: 2}, {baz: 3}];
var nodes = document.getElementsByTagName('div');

debug(isNodeList(array)); // false
debug(isNodeList(nodes)); // true
