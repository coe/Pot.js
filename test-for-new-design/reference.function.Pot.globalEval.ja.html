{*} Pot.globalEval ({String} code)

Pot.js と PotLite.js で利用可能。

グローバルスコープで eval します。

引数 code をグローバルスコープで実行します。
コード内で宣言した関数や変数は、グローバルスコープに影響します。
グローバルに影響しないよう実行したい場合は、かわりに Pot.localEval() が利用できます。

eval と同じく、最後に評価された式または文の結果が返ります。 

globalEval('function hoge() { return "hoge"; }');

debug(hoge()); // 'hoge'
