Pot.Deferred.isFired

{Boolean} Pot.Deferred.isFired ({Pot.Deferred} deferred)

Pot.js と PotLite.js で利用可能。

引数 deferred のチェインが開始されているかどうかの真偽値を返します。

引数 deferred が Pot.Deferred インスタンスであり、すでに開始されている場合は true を返し、
そうでない場合は false を返します。
Pot.globalize() が適応済みの場合、Pot.Deferred.isFired() が isFired() で実行できます。

引数 deferred のチェインが開始されているかどうかの真偽値を返します。 

var d = new Pot.Deferred();

Pot.debug( Pot.Deferred.isFired(d) ); // false

d.then(function() {
    return 'hoge';
});

Pot.debug( Pot.Deferred.isFired(d) ); // false
d.begin();
Pot.debug( Pot.Deferred.isFired(d) ); // true
