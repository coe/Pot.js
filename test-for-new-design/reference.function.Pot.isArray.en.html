{Boolean} Pot.isArray ({*} x)

Pot.js と PotLite.js で利用可能。

引数 x が配列かどうかチェックします。

引数 x が配列かどうか調べ、適正なら true を返します。
配列でない場合、false が返ります。

x が配列なら true を返します。 

debug(isArray([]));            // true
debug(isArray([1, 2, 3]));     // true
debug(isArray(new Array(10))); // true
debug(isArray('hoge'));        // false
