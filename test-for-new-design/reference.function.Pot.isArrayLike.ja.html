{Boolean} Pot.isArrayLike ({*} o)

Pot.js と PotLite.js で利用可能。

引数 o が配列のようなオブジェクトかどうかチェックします。

引数 o が配列のようなオブジェクトかどうか調べます。
基本的に Pot.isArray() と同じです。 違いは、arguments や NodeList も true として返すことです。

o が配列のようなオブジェクトなら true を返し、それ以外は false を返します。 

(function(a, b, c) {
    debug( isArray(arguments) );     // false (ECMA-262 3rd Edition)
    debug( isArrayLike(arguments) ); // true
})(1, 2, 3);
