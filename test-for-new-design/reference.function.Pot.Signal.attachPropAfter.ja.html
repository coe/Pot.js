Pot.Signal.attachPropAfter

{Object|Array} Pot.Signal.attachPropAfter ({Object|Function|String} object, {String|Array} propName, {Function} callback)

Pot.js と PotLite.js で利用可能。

object が持つ関数 propName が呼ばれた後に実行されるシグナル callback を登録します。

attachAfter() と違い、attach() の必要はありません。
登録すると、signal() を呼ばなくても object[propName] が呼ばれた時に自動的に実行されます。
attachPropAfter は、DOM オブジェクトに対しての動作は保証されません。

Pot.globalize() が適応されている場合、Pot.attachPropAfter() が attachPropAfter() で実行できます。

ユニークな ID を保持する ハンドラオブジェクトが返ります。 

// 例えば何らかのアプリケーションを実行する時に、ログをとりたくなった場合
var MyApp = {
    execute : function() {
        // 何らかの処理を開始する
        myAppDoit();
    }
};

Pot.attach('#execute', 'click', function() {
    // アプリケーションを実行
    MyApp.execute();
});

// 実行する前にログを取る
Pot.attachPropBefore(MyApp, 'execute', function() {
    MyLogger.log('Begin execute');
});

// 実行した後のログを取る
Pot.attachPropAfter(MyApp, 'execute', function() {
    MyLogger.log('End execute');
});
