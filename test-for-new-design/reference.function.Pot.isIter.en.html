Pot.isIter

{Boolean} Pot.isIter ({*} x)

Pot.js と PotLite.js で利用可能。

引数 x が Pot.Iter のインスタンスかどうかチェックします。

引数 x が Pot.Iter のインスタンスかどうか調べます。

x が Pot.Iter のインスタンスなら true を返し、それ以外は false を返します。 

var func = function() {};
var iter = new Pot.Iter();

debug( isIter(func) ); // false
debug( isIter(iter) ); // true
