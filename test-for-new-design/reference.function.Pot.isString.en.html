{Boolean} Pot.isString ({*} x)

Pot.js と PotLite.js で利用可能。

引数 x が文字列かどうかチェックします。

引数 x が文字列かどうか調べ、適正なら true を返します。
文字列でない場合、false が返ります。

x が文字列なら true を返します。 

debug(isString(''));                 // true
debug(isString('123'));              // true
debug(isString(new String('hoge'))); // true
debug(isString(123));                // false
