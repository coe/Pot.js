{Boolean} Pot.isWindow ({Window|Element|Node|*} x)

Pot.js と PotLite.js で利用可能。

引数 x が DOM Window オブジェクトかどうかチェックします。

引数 x が DOM Window オブジェクトかどうか調べます。

x が window オブジェクトなら true を返し、それ以外は false を返します。 

// For Web Browser. 
debug(isWindow(window));         // true
debug(isWindow(document));       // false
debug(isWindow(document.body));  // false
