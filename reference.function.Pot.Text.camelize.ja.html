Pot.Text.camelize

{String} Pot.Text.camelize ({String} s)

Pot.js で利用可能。 PotLite.js では利用できません。

文字列 s をキャメルケースにして返します。

引数 s は、対象の文字列を指定します。
キャメルケース (CamelCase) にした文字列が返ります。
細かく言うと、ローワーキャメルケース (LCC) です。

Pot.globalize() が適応されている場合、Pot.camelize() が camelize() で実行できます。

s をキャメルケースにした文字列が返ります。 

var result = Pot.camelize('font-size');
debug(result); // 'fontSize'
