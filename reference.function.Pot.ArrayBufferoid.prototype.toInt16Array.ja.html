Pot.ArrayBufferoid.prototype.toInt16Array

{Int16Array|Array} Pot.ArrayBufferoid.prototype.toInt16Array ()

Pot.js で利用可能。 PotLite.js では利用できません。

Int16Array に変換します。

Pot.ArrayBufferoid 自身が持っている要素と共に Int16Array に変換した新しいオブジェクトを返します。

変換された Int16Array または Array が返ります。 

var bufferoid = new Pot.ArrayBufferoid([1, 2, 3]);
var int16array = bufferoid.toInt16Array();
Pot.debug(int16array); // [1, 2, 3]
