Pot.Workeroid.prototype.postMessage

{Pot.Workeroid} Pot.Workeroid.prototype.postMessage ({String|*} data)

Pot.js と PotLite.js で利用可能。

子 Worker へメッセージ data を送信します。

生成した 子 Worker へメッセージをポストします。
または Worker の開始 として実行できます。
任意に 引数 data に 子 Worker へ送信する文字列が指定できます。

実行した Pot.Workeroid インスタンス自身 が返ります。 

var worker = new Pot.Workeroid(function(data) {
    postMessage(data + 'fuga');
});

// メッセージを受信した時
worker.onmessage = function(data) {
    alert(data);
};

// 'hoge' を 子 Worker に送信 => 'hogefuga' が alert される
worker.postMessage('hoge');
