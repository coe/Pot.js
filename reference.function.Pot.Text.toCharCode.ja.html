Pot.Text.toCharCode

{Array} Pot.Text.toCharCode ({String} string [, {Function} callback])

Pot.js で利用可能。 PotLite.js では利用できません。

文字列 string を Unicode 値の配列に変換します。

文字列 string の各文字をそれぞれ Unicode 値の数値に変換し、その値をもつ配列を返します。
引数 callback が与えられると、各値はその関数の戻り値になります。

Pot.globalize() が適応されている場合、Pot.toCharCode() が toCharCode() で実行できます。

引数 string を Unicode 値の配列にした結果、もしくは callback 通りに変換した結果が返ります。 

var string = 'foo bar ほげ';
var result = Pot.toCharCode(string);
Pot.debug(result);
// [102, 111, 111, 32, 98, 97, 114, 32, 12411, 12370]
var string = 'abc';
var result = Pot.toCharCode(string, function(code) {
    return code.toString(16);
});
Pot.debug(result);
// ['61', '62', '63']
