Pot.Hash.prototype.values

{Array} Pot.Hash.prototype.values ()

Pot.js で利用可能。 PotLite.js では利用できません。

Hash が持っているすべての値を配列で取得します。

Pot.Hash インスタンスに設定された、すべての値を配列で取得します。

すべての値を持つ配列が返ります。 

var hash = new Pot.Hash();
hash.set('foo', {foo: 1});
hash.set('bar', ['bar', 2]);
hash.set('baz', function() { return 'baz'; });
hash.set('qux', 3);
hash.set('quux', 3);

Pot.debug(hash.values());
// [{foo: 1}, ['bar', 2], function() { return 'baz'; }, 3, 3]
hash.clear();
Pot.debug(hash.keys()); // []
