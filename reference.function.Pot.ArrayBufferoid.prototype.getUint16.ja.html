Pot.ArrayBufferoid.prototype.getUint16

{Number} Pot.ArrayBufferoid.prototype.getUint16 ([{Number} byteOffset [, Boolean littleEndian = false]])

Pot.js で利用可能。 PotLite.js では利用できません。

Uint16 で値を取得します。

Pot.ArrayBufferoid インスタンスの現在の位置から Uint16 として値を取得します。
引数 byteOffset を指定すると、位置が指定できます。
引数 littleEndian を 真 (true) として渡すと、リトルエンディアンで返ります。
この関数は DataView と同様に扱えます。

取得した Uint16 の値が返ります。 

var buffer = new Pot.ArrayBufferoid([
  72, 101, 108, 108, 111, 32, 87, 111, 114, 108, 100
]);
Pot.debug(buffer.getUint16()); // 25964
