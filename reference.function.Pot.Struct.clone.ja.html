Pot.Struct.clone

{*} Pot.Struct.clone ({*} o)

Pot.js で利用可能。 PotLite.js では利用できません。

引数 o のクローンを返します

引数 o のクローンを作成し、それを返します。

Pot.globalize() が適応されている場合、Pot.clone() が clone() で実行できます。

o のクローンが返ります。 

var obj1 = {key: 'value'};
var obj2 = Pot.clone(obj1);
obj2.hoge = 'fuga';
Pot.debug(obj1.hoge);  // undefined
Pot.debug(obj2.hoge);  // 'fuga'
