Pot.System.isNonBrowser

Pot.System.isNonBrowser {Boolean|Void}

Pot.js と PotLite.js で利用可能。

Web ブラウザ以外で実行されているかどうか。

Web ブラウザ以外で実行中なら、Pot.System.isNonBrowser が true になります。
違う場合、Pot.System.isNonBrowser は undefined になります。
例えば Node.js での実行が該当します。 

if (Pot.System.isNonBrowser) {
    // Web ブラウザ以外で実行されている
    // ...
}
