Pot.System.isFirefoxExtension

Pot.System.isFirefoxExtension {Boolean|Void}

Pot.js と PotLite.js で利用可能。

Mozilla Firefox Extension (AddOn) で実行されているかどうか。

Mozilla Firefox Extension (AddOn) で実行中なら、Pot.System.isFirefoxExtension が true になります。
違う場合、Pot.System.isFirefoxExtension は undefined になります。 

if (Pot.System.isFirefoxExtension) {
    // Mozilla Firefox Extension で実行中
    // ...
}
