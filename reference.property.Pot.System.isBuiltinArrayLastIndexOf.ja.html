Pot.System.isBuiltinArrayLastIndexOf

Pot.System.isBuiltinArrayLastIndexOf {Boolean|Void}

Pot.js と PotLite.js で利用可能。

Array.prototype.lastIndexOf がビルトインでサポートされているかどうか。

Array.prototype.lastIndexOf が、
ビルトインでサポートされている場合 Pot.System.isBuiltinArrayLastIndexOf が true になります。
サポートされていない場合、false になります。
実行後に Array.prototype.lastIndexOf = function() {...} と定義しても false になります。
Array.prototype.lastIndexOf は、JavaScript 1.6 以降でサポートされます。 

if (Pot.System.isBuiltinArrayLastIndexOf) {

    // Array.prototype.lastIndexOf がサポート済み
    debug( ['a', 'b', 'c', 'b', 'd'].lastIndexOf('b') ); // 3

}
