Pot.Sanitizer.unescapeHTML

{String} Pot.Sanitizer.unescapeHTML ({String} text)

Pot.js で利用可能。 PotLite.js では利用できません。

引数 text を HTML 文字列としてアンエスケープします。

引数 text をエスケープ済みの HTML 文字列として、アンエスケープして返します。
基本的に Pot.escapeHTML の逆の動作をします。
また、他のエンティティも変換します。
例えば &apos; は ' (U+0027) になり、&middot; は ・ (U+30FB) になります。

Pot.globalize() が適応されている場合、Pot.unescapeHTML() が unescapeHTML() で実行できます。

アンエスケープされた文字列が返ります。 

var string = '(&gt;_&lt;)/&quot;&lt; Hello World!';
var result = Pot.unescapeHTML(string);
Pot.debug(result);
// '(>_<)/"< Hello World!'
