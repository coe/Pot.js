Pot.Struct.glue

{String} Pot.Struct.glue ({Array|...*} args)

Pot.js で利用可能。 PotLite.js では利用できません。

引数を 1 つの文字列に連結して返します。

引数は可変で指定するか、配列で渡します。
すべてを連結した 1 つの文字列が返ります。

Pot.globalize() が適応されている場合、Pot.glue() が glue() で実行できます。

引数すべてを 1 つに連結した文字列が返ります。 

Pot.debug(Pot.glue([1, 2, 3, 4, 5]));
// '12345'

Pot.debug(Pot.glue('foo', 'bar', 'baz'));
// 'foobarbaz'

Pot.debug(Pot.glue(1, [2, 3, ['foo']], ['bar', 'baz']));
// '123foobarbaz'
