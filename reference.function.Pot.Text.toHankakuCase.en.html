Pot.Text.toHankakuCase

{String} Pot.Text.toHankakuCase ({String} text)

Pot.js で利用可能。 PotLite.js では利用できません。

全角英数記号文字を半角英数記号文字に変換します。

引数 text は、変換する文字列を指定します。

Pot.globalize() が適応されている場合、Pot.toHankakuCase() が toHankakuCase() で実行できます。

全角英数記号文字を半角英数記号文字に変換した文字列が返ります。 

debug(Pot.toHankakuCase('Ｈｅｌｌｏ Ｗｏｒｌｄ！ １２３４５'));
// 'Hello World! 12345'
