Pot.Struct.reverse

{Object|*} Pot.Struct.reverse ({Object|Array|*} o)

Pot.js で利用可能。 PotLite.js では利用できません。

引数 o の値を逆にして返します。

値を逆にした 引数 o のコピーを取得します。

Pot.globalize() が適応されている場合、Pot.reverse() が reverse() で実行できます。

値を逆にした o のコピーが返ります。 

Pot.debug(Pot.reverse({foo: 1, bar: 2, baz: 3})); // {baz: 3, bar: 2, foo: 1}
Pot.debug(Pot.reverse([1, 2, 3, 4, 5])); // [5, 4, 3, 2, 1]
Pot.debug(Pot.reverse('123abc')); // 'cba321'
