Pot.System.hasActiveXObject

Pot.System.hasActiveXObject {Boolean|Void}

Pot.js と PotLite.js で利用可能。

ActiveXObject がサポートされているかどうか。

ActiveXObject がサポートされている場合 Pot.System.hasActiveXObject が true になります。
サポートされていない場合、false になります。
ActiveXObject は Microsoft Internet Explorer などでサポートされています。 

if (Pot.System.hasActiveXObject) {

    // ActiveXObject が利用できる
    var xhr = new ActiveXObject('Microsoft.XMLHTTP');
    // ...

}
