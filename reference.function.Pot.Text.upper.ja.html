Pot.Text.upper

{String} Pot.Text.upper ({String} s)

Pot.js で利用可能。 PotLite.js では利用できません。

文字列 s をすべて大文字にして返します。

upper は、String(s).toUpperCase(); のショートカットです。
ただし、s が文字列でない場合は stringify() によって文字列として扱います。

Pot.globalize() が適応されている場合、Pot.upper() が upper() で実行できます。

s をすべて大文字にした文字列が返ります。 

debug(upper('Hello World!')); // 'HELLO WORLD!'
