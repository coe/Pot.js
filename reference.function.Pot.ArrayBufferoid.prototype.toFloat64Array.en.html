Pot.ArrayBufferoid.prototype.toFloat64Array

{Float64Array|Array} Pot.ArrayBufferoid.prototype.toFloat64Array ()

Pot.js で利用可能。 PotLite.js では利用できません。

Float64Array に変換します。

Pot.ArrayBufferoid 自身が持っている要素と共に Float64Array に変換した新しいオブジェクトを返します。

変換された Float64Array または Array が返ります。 

var bufferoid = new Pot.ArrayBufferoid([1, 2, 3]);
var float64array = bufferoid.toFloat64Array();
Pot.debug(float64array); // [1, 2, 3]
